# 暮色 · Twilight

一张冬日黄昏的照片，一段关于时间与记忆的旅程。

GTA6 风格沉浸式滚动视觉体验。

## ✨ 特性

- 🎬 GSAP 驱动的滚动动画
- 🌌 Canvas 粒子特效
- 📐 多层视差效果
- 🖱️ 自定义光标交互
- 📱 响应式设计

## 🚀 预览

直接打开 `index.html` 即可在浏览器中查看。

## 🛠️ 技术栈

- HTML5 / CSS3 / Vanilla JavaScript
- [GSAP](https://greensock.com/gsap/) + ScrollTrigger
- Google Fonts (Inter / Noto Serif SC)
